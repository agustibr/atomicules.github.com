Webkit browsers are really kicking Gecko based one's at the moment. Both in the rendering engine speed and in javascript speed. So this is good to see. I managed to get this working in Slax using the deb2lzm module and installing these other two modules: libgconf2-4 and Liborbit2. It crashes like mad, which could either me something else I've missed in Slax or just that this is a very early developer release.

As far as OSX goes, there's a lot of people moaning about lack of PPC support, but PPC is dead so there really is no point developing for it. It's a shame, but there are alternatives.
